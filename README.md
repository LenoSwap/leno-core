In-depth documentation on LenoSwap is available at [docs.lenoswap.com](https://docs.lenoswap.com/).

# Local Development

The following assumes the use of `node@>=10`.

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`
